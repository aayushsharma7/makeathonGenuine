+# OpenCourse - Hackathon Brief                                                                                                                     
     2 +                                                                                                                                                   
     3 +## Team                                                                                                                                            
     4 +**Team Name:** Genuine                                                                                                                             
     5 +                                                                                                                                                   
     6 +## Problem                                                                                                                                         
     7 +High-quality learning content is freely available on YouTube, but learners struggle with:                                                          
     8 +- No structure across long playlists                                                                                                               
     9 +- No personalized learning path                                                                                                                    
    10 +- Weak progress tracking and revision discipline                                                                                                   
    11 +- High drop-off before course completion                                                                                                           
    12 +                                                                                                                                                   
    13 +## Solution                                                                                                                                        
    14 +OpenCourse converts YouTube playlists into personalized, structured learning journeys with AI guidance, adaptive assessment, and cross-platform con
        tinuity.                                                                                                                                           
    15 +                                                                                                                                                   
    16 +## Key Features                                                                                                                                    
    17 +- Dual-path onboarding:                                                                                                                            
    18 +  - Learner already knows the course (playlist import)                                                                                             
    19 +  - Learner needs AI recommendation                                                                                                                
    20 +- AI-powered module generation (topic-wise course structuring)                                                                                     
    21 +- Personalized video recommendations (watch/skip/speed/summary-first)                                                                              
    22 +- AI tutor + AI video overview + generated summaries                                                                                               
    23 +- Adaptive quiz system with retake and concept-focused evaluation                                                                                  
    24 +- Timestamped smart notes with categories and spaced revision workflow                                                                             
    25 +- Dynamic planner with daily targets and progress insights                                                                                         
    26 +- Built-in Monaco IDE and Excalidraw whiteboard in course player                                                                                   
    27 +- Chrome extension with real-time sync between YouTube and website                                                                                 
    28 +                                                                                                                                                   
    29 +## Tech Stack                                                                                                                                      
    30 +- **Frontend:** React, Vite, Tailwind CSS, Monaco Editor, Plyr                                                                                     
    31 +- **Backend:** Node.js, Express.js, MongoDB, JWT, Bcrypt
    32 +- **AI / RAG:** Groq, Pinecone                                                                                                                     
    33 +- **Extension:** Chrome Extension (Manifest V3)                                                                                                    
    34 +- **External APIs:** YouTube Data API v3, Piston API, YouTube transcript APIs                                                                      
    35 +                                                                                                                                                   
    36 +## Why It Matters                                                                                                                                  
    37 +OpenCourse turns passive video consumption into an active, measurable, and personalized learning system that improves completion, consistency, and 
        retention.
